# MP3 Tag Reader project

## Project Description:
MP3 Tag Reader is a C programming project used to read and edit MP3 metadata. 
It extracts information from MP3 files and displays details such as song title, artist, album, year, and genre using ID3 tag format.

## Technologies Used
- C Programming
- File Handling
- Structures
- Pointers
- Binary File Operations

## Features
- View MP3 details
- Display song title
- Display artist name
- Display album name
- Display year
- Edit MP3 tag information

## Concepts Used
- File I/O
- Structures
- Command line arguments
- String handling
